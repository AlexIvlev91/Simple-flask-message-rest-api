####For post request body looks like this: 
```
{
    "receiver":"string",
    "message":"string",
    "subject":"string",
    "id":"string",
    "owner":"string"
}
```

####Post parameters:
```
username = username
password = password
```

####Get parameters:
```
username = username
password = password
id = None or Id of massage(None for get all and get unread message )
receiver = name of receiver
action = all/read/unread
```

####Get delete:
```
username = username
password = password
id = None or Id of massage(None for get all and get unread message )
```
